Overview
This project compares three machine learning models for classifying images of religious symbols (Christianity, Islam, and Judaism):

Naive Bayes — 43% accuracy
Decision Tree — 72% accuracy (best performer)
MLP Neural Network — 30% accuracy

Dataset
654 images scraped from Google, organized into 3 categories, resized to 64×64 grayscale pixels and flattened into 1D vectors.
